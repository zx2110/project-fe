## 1일차 작업 내용

### 작업 개요
- React와 Vite를 이용한 AI 가계부 프로젝트 초기 작업 진행
- 대시보드 화면 구현 및 사이드바 추가
- Git을 이용한 버전 관리 및 커밋 작업 진행

### 세부 내용

1. **대시보드 화면 구성**
   - 기본 대시보드 화면 구현
   - '영수증 추가하기' 버튼 추가

2. **사이드바 기능 추가**
   - 사이드바에 '카테고리별 지출내역' 메뉴 항목 추가
   - 메뉴 항목들이 세로로 정렬되도록 레이아웃 수정

3. **Git 커밋 및 관리**
   - 변경된 파일들 `git add .`로 스테이징 완료
   - 줄바꿈(LF/CRLF) 관련 경고 확인 및 설명
   - 커밋 명령어 및 푸시 과정 안내
